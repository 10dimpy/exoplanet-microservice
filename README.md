# Exoplanet Microservice

This is a simple Golang microservice to manage exoplanets.

## Features

- Add an Exoplanet
- List Exoplanets
- Get Exoplanet by ID
- Update Exoplanet
- Delete Exoplanet
- Estimate Fuel for a trip to an Exoplanet

## Requirements

- Go 1.19 or later
- Docker
